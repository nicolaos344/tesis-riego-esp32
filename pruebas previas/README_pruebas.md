
Incluyen:
- Prueba básica del microcontrolador (Blink).
- Prueba de display OLED (descartado).
- Pruebas con sensores DHT11 y DHT22.
- Pruebas de conexión WiFi.
- Pruebas de comunicación HTTPS con base de datos remota.
